Distro Linux Terbaik Untuk Developer di Tahun 2021

Distro Linux terbaik untuk developer  yang mudah, stabil, dan aman untuk scripting dan pemrograman aplikasi untuk internet, Android, dan cloud.

Ini semakin penting, karena Linux mendukung tulang punggung internet, perangkat seluler, dan sekarang sistem Cloud Computing.

Oleh karena itu, seringkali para developer handal sudah terbiasa memulai projectnya di atas Linux, terutama untuk mengoperasikan server dan untuk mengembangkan perangkat lunak yang berjalan di atasnya.

Sementara Linux memiliki reputasi sebagai pembuat kode dan pemrogram, selama beberapa dekade terakhir telah ada langkah untuk menyediakan versi Linux yang lebih ramah bagi penggunanya, seperti dengan menyediakan lebih banyak graphic user interface  (GUI) dan kurang bergantung pada penggunaan command line.

Namun, pada intinya Linux masih menawarkan distro linux yang berkembang berkembang untuk scripting dan developer.

Berikut adalah sistem operasi Linux terbaik yang akan membantu sobat teknomid membuat custom programming  dan development workstation sobat.

1. Manjaro

Kenapa harus manjaro ....? , di bawah ini akan kita coba paparkan kenapa manjaro masuk sebagai salah satu Distro Linux untuk Developer.

Manjaro adalah baseon Arch Linux dimana system ini sangatlah bisa di custom sendiri sesuai dengan keinginan para developer system maupun applikasi.

Baca Juga: Manjaro Linux 2021(edited)

Sekilas kita coba cari tahu dulu apa itu arch linux,
Arch Linux adalah sebuah distribusi Linux berbasis biner, yang artinya, paket-paket aplikasi didistribusikan dalam bentuk yang telah terkompilasi. Distro ini pendekatannya dengan desain pengembang, distro ini berfokus pada kesederhanaan, kebenaran program dan minimalisme.

Arch Linux sangatlah powerful yang bisa kita gunakan dan membuat customized  untuk environment kita sendiri. Namun menginstall Arch Linux ini sangatlah melelahkan, dan MANJARO menawarkan Distronya dengan GUI frendly.

Manjaro menyediakan dalam beberapa edisi dengan desktop environment yang berbeda. Jika sobat adalah developer Qt, Anda dapat menggunakan distro edisi KDE, yang disertakan dengan tools seperti Qt Designer dan Qt Assistant untuk membantu Anda dalam development.

Manjaro mengklaim distro tersebut dilengkapi dengan tools yang diperlukan untuk compile dan develop software untuk developer. Di manjaro ini Anda dapat menggunakan manajer paket intuitif Manjaro untuk menginstal IDE dengan mudah seperti Qt Creator, KDevelop atau Netbeans dan library seperti libnoise, boost, atau matplotlib. Tentu saja selain repositorinya sendiri, Anda juga dapat mengakses Arch User Repository (AUR) yang komprehensif untuk menampung semua jenis tools dan library yang kita butuhkan.

2. Puppy Linux

Puppy Linux adalah distro yang sangat ringan yang memiliki edisi terpisah berdasarkan Ubuntu dan Slackware. Seluruh OS cukup kecil untuk dijalankan sepenuhnya dalam RAM, yang membuatnya sangat cepat dan responsif. Anda juga dapat menambatkan distro ke hard disk Anda.

Meskipun ukurannya kecil, tidak ada yang mengalahkan Puppy untuk box functionality. Distro dikirimkan dengan aplikasi dan utilitas untuk hampir semua fungsi yang dapat Anda lakukan di desktop.

Ada beberapa mekanisme untuk menginstal aplikasi di Puppy. Salah satu yang paling nyaman melibatkan file SFS (SquashFS), yang merupakan lingkungan terkompresi yang mengemas satu atau lebih aplikasi dan semua dependensi yang diperlukan. Jika Anda ingin mengembangkan di Puppy Anda dapat menggunakan file devx SFS yang berisi berbagai tools development dan build.

Selain itu, Puppy Linux memiliki pengantar yang bagus untuk pemrograman, yang merupakan titik awal yang baik untuk pengembang baru. Halaman ini juga menunjukkan cara menginstal dukungan untuk lebih dari selusin bahasa pemrograman dalam instalasi Puppy Anda.

3. Solus

Solus istimewa karena merupakan salah satu dari sedikit distro Linux Irlandia, dan juga karena mengikuti model rilis rolling yang dikuratori. Keuntungannya adalah setelah Anda menginstal OS, Anda dapat terus menjalankan pembaruan daripada peningkatan besar-besaran. Solus, bagaimanapun, mencoba untuk menghindari menginstal paket yang sangat baru dan perangkat lunak beta untuk menjaga stabilitas sistem.

Distro mengklaim penggunaannya sebagai development yang ideal untuk pengembang. Solus mendukung beberapa editor dan IDE seperti Atom, Idea dan Gnome Builder, serta beberapa sistem kontrol versi termasuk Git, Bazaar, dan lainnya melalui alat grafis seperti GitKraken dan git-cola. Situs web proyek Solus juga mengklaim bahwa OS tersebut mendukung sejumlah bahasa pemrograman seperti Go, Rust, PHP, Node.js, dan Ruby.

Jadi sementara distro mungkin tidak dikirimkan dengan "very many tools out of the book", Anda dapat dengan mudah menyempurnakannya  development anda dengan toolschain.

4. Ubuntu

Ubuntu adalah salah satu distro Linux paling populer untuk semua jenis pengguna, dari pemula Linux hingga juru kampanye berpengalaman. Untuk programmer, rilis Ubuntu Long Term Support (LTS) menyediakan lingkungan pengembangan yang stabil sehingga mereka tidak perlu memutakhirkan setiap enam bulan.

Faktanya, Ubuntu adalah distro yang dipilih dari Android Open Source Project untuk membangun sources files. Build Android diuji secara rutin menggunakan versi Ubuntu terbaru. Situs web resmi distro menyelenggarakan beberapa tutorial, panduan, dan sumber daya lainnya yang bertujuan untuk mengesankan pengembangan dan kecakapan pemrograman distro.

Karena popularitasnya, Anda dapat menemukan hampir semua alat dan pustaka pengembangan dan pemrograman di repositori resmi Ubuntu atau di Arsip Paket Pribadi (PPA). Dengan diperkenalkannya format kemasan snap, menginstal perangkat lunak baru adalah proses yang mudah.

Selain itu, Ubuntu memiliki developer friendly command-line yaitu tools ramah pengembang yang disebut Ubuntu Make yang dapat Anda gunakan untuk mengunduh beberapa alat ramah pengembang.
