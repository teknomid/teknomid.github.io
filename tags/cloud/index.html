CLOUD (1)

"Cloud" mengacu pada server yang diakses melalui Internet, dan perangkat lunak serta database yang berjalan di server tersebut. Server cloud berlokasi di pusat data di seluruh dunia. Dengan menggunakan Cloud Computing, pengguna dan perusahaan tidak perlu mengelola server fisik sendiri atau menjalankan aplikasi perangkat lunak di mesin mereka sendiri.
