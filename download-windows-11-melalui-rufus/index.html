Download Windows 11 Melalui Rufus

teknomid October 15, 2021

Windows 11 baru-baru ini sangat banyak di perbincangkan di dunia dan rufus mencoba langsung update meta download ISO jagoan windows satu ini.

Pada versi Rufus terbaru, kamu sudah langsung dapat mendownload file ISO Windows-11 dari berbagai versi Windows untuk selanjutnya kamu simpan dan gunakan nanti.

Selain kamu dapat mendownload file instalasi Windows 11 secara langsung dari halaman Microsoft, kamu juga dapat mendownloadnya di Rufus loh, dan langkahnya sangat sederhana, berikut WinPoin rangkum langkah singkatnya.

  1. Langkah-Pertama

    Silahkan kamu download versi Rufus terbaru terlebih dahulu (kamu bisa mendownloadnya disini)

  2. Langkah-Kedua

    Silahkan kamu buka Rufus yang sudah di download dan klik pada tombol panah Download disamping Boot selection.

  3. Langkah-Ketiga

    Langkah Selanjutnya, silahkan kamu klik tombol Download dan Pilih Windows 11

  4. Langkah-KeEmpat

    Di Tahap Selanjutnya akan di arahkan memilih Version, Release, Edition, Language, Architecture, (Rekomendasi Ceklist Download Using a Browser agar lebih mudah dan bisa langsung native download lewat IDM) dan Click Download

  5. Langkang-KeLima

    Langkah ini akan di bawakan ke Drive atau tempat dimana akan menyimpan File ISO Windows 11 yang akan kita Proses Download
    Baca-Juga: Applikasi Desain Gratis

    dan tunggu hingga selesai

  6. Langkah-KeEnam

    Progress Download File ISO Melalui Browser tanpa IDM

Proses Selanjutnya kita sudah bisa membuat bootable installer Windows 11 dengan Rufus lewat USB Drive ataupun CD-Drive.

  • Mengatasi Whatsapp yang sudah Kadaluarsa
  • Perintah Dasar Linux
  • DebianOS "Kenali Dahulu Baru Sayangi"
  • Aplikasi Desain di Linux dan Windows Open Source

Step By Step

1. Hubungkan perangkat flash disk atau penyimpanan sejenis. Pastikan letter drive telah sesuai dengan target bootable.
2. Klik opsi SELECT dan pilih file ISO Windows 11 yang kamu miliki.
3. Pilih opsi antara Standard Windows Installation atau Windows To Go. Jika ingin melakukan installasi biasa, maka pilih Standard Windows Installation.
4. Pilih GPT.
5. Bergantung pada langkah ke-4. Lebih lanjut penjelasan bisa kamu lihat pada bagian bawah.
5. Untuk bagian Name Kamu bisa menggantinya atau membiarkannya default.
Karena Windows 11 hanya mendukung UEFI, maka pilih GPT
Silahkan tunggu proses pembuatan bootablenya.
